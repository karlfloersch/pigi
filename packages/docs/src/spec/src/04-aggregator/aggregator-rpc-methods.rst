pgop_sendTransaction
====================

Description
^^^^^^^^^^^
Sends a transaction to be `ingested`_ by the aggregator.

Parameters
^^^^^^^^^^
1. ``transaction`` - ``Transaction``: The `Transaction`_ to be ingested.

Returns
^^^^^^^
``string``: The `transaction receipt`_ for the given transaction.
